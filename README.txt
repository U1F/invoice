== Description ==

= Q Invoice =

Q Invoice is a new plugin to organize and create invoices on your website using Wordpress.

Your invoice can be created and managed easily in the backend.


**Quick Facts**
The plugin has all you need to create invoices:
* recipient list / customer list
* create / edit invoices
* download PDFs
* managing of invoice details


**Overview Back End**
On the back end a list of all invoices is shown. Via one click a pop up will show up and a new invoice can be inserted. Same for updating an event.
The user can enter all details. 
Same for update.
The toggle "Status" steers if an event is active, planned or a Storno was created.


**Overview Front End**
No Frontend
